# Michael Jackson | The King of Pop – Tribute Page

## Popis projektu
Tento projekt je moderní a vizuálně atraktivní web věnovaný Michaelu Jacksonovi. Web slouží jako náhled do jeho života, největšími kariérními milníky, ikonickou diskografií a nesmrtelným odkazem. 


## Použité technologie a knihovny

### Core
- **HTML5**: Sémantická struktura stránky a SEO optimalizace.
- **Vanilla CSS3**: Pokročilý styling, responzivní layout, skleněný efekt animace.
- **Vanilla JavaScript: Logika ovládání menu, loaderu a interaktivních prvků.

### Knihovny a API
- **[GSAP (GreenSock Animation Platform)](https://greensock.com/gsap/)**: Použito pro plynulé úvodní animace, pohybové efekty při scrollování.
- **[GSAP ScrollTrigger](https://greensock.com/scrolltrigger/)**: Zajišťuje spouštění animací přesně ve chvíli, kdy se prvek objeví na obrazovce.
- **Google Fonts**: Typografie využívající fonty Cinzel (nadpisy) a Montserrat (text).

## Návod na spuštění
Není potřeba nic extra.

1. Stáhněte nebo naklonujte složku s projektem.
2. Otevřete soubor `Michael Jackson.html` v libovolném moderním webovém prohlížeči.

## Prvky, na které jsme nejvíce hrdí
- **Moonwalk Animation**: Interaktivní sekce "The Moonwalk", kde se pomocí GSAP animují stopy bot v nekonečné smyčce, což má symbolizovat Michaelův ikonický taneční pohyb.
- **Dynamické počítadlo statistik**: V sekci "Síň Slávy" se při nascrollování automaticky spustí animovaný odpočet rekordů.
- **Karty diskografie**: Interaktivní karty alb s efektem převrácení.
